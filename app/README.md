# LogSense

LogSense is a FastAPI-based log analysis tool that processes `.log` files and extracts structured insights such as errors, severity levels, and priorities.

## Features
- Upload and analyze log files
- Extract errors and warnings
- Severity & priority classification
- Handles empty log files
- REST API based architecture

## Tech Stack
- Python 3.11
- FastAPI
- Uvicorn
